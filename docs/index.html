About the App

Assignment Tracker helps students manage their academic workload with ease. You can add or edit classes with details like title, subject, start date, and end date. For each class, you can add or edit assignments with details such as title and due date.

Features include a search bar to quickly find classes, sharing options for classes and assignments, and CSV report generation for all your assignments.

Developed in Android Studio using Java and Room SQL as part of a Software Engineering Capstone Project.
